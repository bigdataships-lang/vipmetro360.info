Customization (5 Minutes)

1. Change Colors

Open index.html, find the <style> section:

:root { --primary-neon: #00ff88; /* Main color */ --secondary-neon: #ff006e; /* Secondary */ --tertiary-neon: #00d4ff; /* Accent */ --gold: #ffd700; /* Highlights */ }

2. Change Game Prices

Open app.js, find GAME_CONFIG:

GAME_CONFIG = { game1: { price: 10 }, // ‚Çπ10 game2: { price: 100 } // ‚Çπ100 }

3. Change Prizes

Find calculatePrize() function in app.js:

const prizes = [0, 50, 100, 500, 1000, 5000, ...]; // Modify values as needed

4. Add Your Logo/Branding

In index.html, modify the header section to add your branding

5. Enable Real Razorpay

In app.js, add your API key:

key: 'rzp_live_YOUR_API_KEY_HERE'
